# NeetCode Solutions — @etiennevzn

## What is this?

[NeetCode.io](https://neetcode.io) is a coding interview preparation platform featuring curated problems, video solutions, and an in-browser code editor. This repository is automatically populated with my accepted solutions.

---

## Repository structure

Solutions are organized by topic folder, then problem ID. Each submission is stored as a separate file:

```
<topic-folder>/
  <problem-id>/
    submission-0.<ext>   ← first submission
    submission-1.<ext>   ← second submission
    ...
```
